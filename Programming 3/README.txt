Programming Assignment 3 - Library

Files in this project:
  - LibraryMain.java
  - BookType.java
  - Patron.java
  - bookData2.txt

How to compile (from terminal):
  javac LibraryMain.java BookType.java Patron.java

How to run:
  java LibraryMain

IDE used:
  IntelliJ IDEA Community Edition 2024

Notes:
  This program loads book data from bookData2.txt and allows:
    - printing all books
    - printing available books
    - searching
    - adding patrons
    - checking out books
    - checking in books
    - printing patrons + checked-out books
